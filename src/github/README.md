The following command will generate a `.github/workflows/drizzlekit-migrate.yml` file in your project:

```bash
fluentci gh init -t drizzlekit_pipeline
```

Or, if you already have a `.fluentci` folder (generated from `fluentci init -t drizzlekit`) in your project:

```bash
fluentci gh init
```

Generated file:

```yaml
# Do not edit this file directly. It is generated by Fluent Github Actions

name: DrizzleKit
on:
  push:
    branches:
      - main
jobs:
  apply:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: denoland/setup-deno@v1
        with:
          deno-version: v1.37
      - name: Setup Fluent CI CLI
        run: deno install -A -r https://cli.fluentci.io -n fluentci
      - name: Setup Dagger
        run: |
            curl -L https://dl.dagger.io/dagger/install.sh | DAGGER_VERSION=0.8.1 sh
            sudo mv bin/dagger /usr/local/bin
            dagger version
      - name: Run Dagger Pipelines
        run: fluentci run drizzlekit_pipeline
        env:
          DATABASE_URL: ${{ secrets.DATABASE_URL }}
```

Feel free to edit the template generator at `.fluentci/src/github/config.ts` to your needs.
